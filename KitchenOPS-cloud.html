⚠️ CONFIG MISSING

Πριν χρησιμοποιήσεις την εφαρμογή, πρέπει να συμπληρώσεις τα δικά σου Supabase credentials στην αρχή του αρχείου HTML.

Άνοιξε το αρχείο με έναν text editor και βρες αυτές τις γραμμές:

const SUPABASE_URL = '...'; const SUPABASE_ANON_KEY = '...';

Αντικατάστησε τις τιμές με αυτές από το Supabase Dashboard → Settings → API.
